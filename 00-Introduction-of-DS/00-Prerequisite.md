# Flowchart

Flowchart is the diagram of a solution.

We use some components to draw this diagram.

- Oval
- Parallelogram
- Rectangle
- Arrow
- Diamond shape

&nbsp;

&nbsp;

| Process               | Components    |
| --------------------- | ------------- |
| Start, Stop           | Oval          |
| Input, output         | Parallelogram |
| Process / Calculation | Rectangle     |
| Connector             | Arrow         |
| Decision              | Diamond       |
|                       |               |
