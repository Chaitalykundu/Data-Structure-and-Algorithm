# Problem statement

Write a program to count the number of 1's in the binary representation of an integer.

&nbsp;

# Detailed explanation ( Input/output format, Notes )

## Constraints

1 <= N <= 100

```md
Sample Input 1:
9
Sample Output 1:
2

Explanation of Sample Input 1:
Binary Representation of 9 is 1001.

Sample Input 2:
13
Sample Output 2:
3
```
